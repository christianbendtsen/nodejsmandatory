Node.js

Node.js er et open-source, cross-platform system som gør det muligt at bruge JavaScript server-side.

Det er baseret på Google’s Chrome V8, som gør det utroligt hurtigt og pålideligt.

JavaScript er i stand til at foretage en masse asynkrone kald, hvilket gør at det kan færdiggøre opgaver sideløbende med hinanden. Dette gør ydeevnen meget hurtigere end de fleste nuværende populære sprog, som f.eks. Ruby, Python, Perl, PHP og ASP.NET.

Opsætnig Express.js

Express er et webapp-framework. Det er pt. det mest populære framework til Node.js og bliver brugt i stort set alt vores udvikling med Node.js.
